# حساباتي — نماذج ملفات الاستيراد

ملفات Excel اصطناعية للاختبارات تطابق تنسيقات تصدير **حساباتي** الشائعة.

## الأنواع المدعومة

| الملف | الأعمدة المتوقعة |
|-------|------------------|
| `(3) قائمة الأسعار` | باركود، الاسم، مفرق، جملة (اختياري)، نصف جملة (اختياري) |
| `(2) أرصدة الموردين` | الرقم، اسم المورد، الهاتف (اختياري)، الرصيد |
| `(4) أرصدة زبون` | الرقم، اسم الزبون، الهاتف (اختياري)، الرصيد |
| `أرصدة المشغلين` | نفس أرصدة الزبون — يُستورد كعملاء مع وسم «مشغل» |
| `(5) أرصدة العمارة` | نفس أرصدة الزبون — يُستورد كعملاء مع فئة «شركات» ووسم «عمارة» |

**ملاحظة:** ملف الاختصار (`.lnk`) غير مدعوم — صدّر `.xlsx` حقيقي من حساباتي.

## ترتيب الاستيراد الموصى به

1. بطاقة الأصناف (منتجات + باركود)
2. قائمة الأسعار
3. أرصدة الموردين
4. أرصدة زبون / عمارة / مشغلين
